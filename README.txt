Простой тренировочный проект с достаточно скудным функционалом - вы можете задать параметры вашего пароля (цифры, строчный/прописные буквы, символы). Программа выведет вам готовый случайно сформированный пароль.
Имеется небольшое визуальное оформление в конце выполненное стандартными ANSI кодами

A simple training project with a rather meager functionality - you can set the parameters of your password (numbers, lowercase/uppercase letters, symbols). The program will output you a ready-made randomly generated password.
There is a small visual design at the end made with standard ANSI codes.
